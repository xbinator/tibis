# 2026-06-09

## Added
- 新增跨平台图片预览入口：macOS 使用系统原生图片预览，其他平台使用独立预览窗口。

## Changed
- 聊天输入区、消息气泡和 Markdown 消息图片点击改为直接调用图片预览 hook，原生预览失败时由 hook 内部回退到应用内图片查看器。
- macOS 远程图片预览改为先下载到临时文件再交给系统预览，下载失败时回退到独立预览窗口。
- 图片预览临时文件支持按来源复用缓存，过期缓存会重新下载，无扩展名 URL 可按来源 hash 命中缓存，并在写入新文件前主动清理超过 24 小时的旧缓存。
- 远程图片下载新增 15 秒超时、50MB 体积限制和同 URL 并发下载去重。
- 图片预览 hook 内部统一规范化起始位置并吞掉预览异常，业务调用方不再处理索引兼容或错误兜底。
