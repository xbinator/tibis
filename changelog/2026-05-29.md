# 2026-05-29

## Changed
- BEditor Rich 大文档解析增加最近文档缓存，切回未变更的大文档时复用已解析 JSON，减少重复 Markdown 解析卡顿。

## Fixed
- 修复 Skill 工具概率性不发送的问题：`allBuiltinTools` 在 `<script setup>` 阶段一次性创建时 `skillStore.initialized` 为 `false`，导致 `createBuiltinTools` 中 skill 工具条件检查失败、工具被永久丢弃。改为在 `getActiveTools()` 中每次动态检查 `skillStore.initialized` 并补入 skill 工具。
- 修复 Shell 命令在 skill 目录执行时 cwd 越界被拒的问题：`getAdditionalShellWorkspaceRoots` 闭包捕获了工具创建时的 `enabledSkills` 静态快照（为空），改为每次调用时从 store 动态读取。
- 优化 Shell 命令确认体验：安全分析无 risk finding 时直接放行（如 `ls`、`echo` 等安全命令），有 finding 时才弹窗让用户确认。
