# 2026-06-27

## Changed
- 将 BDrawing 文本元素正文迁移到元素 `metadata.content`，创建时从元素 schema 写入默认 metadata。
- 将 BDrawing 元素背景、边框、字体和对齐样式收敛到 `DrawingNode` 统一渲染，减少元素视图重复样式。
- 为 BDrawing 元素 schema 增加统一默认样式配置，创建元素时写入背景、边框、字体和对齐默认值。
- 为 BDrawing 元素 schema 增加缩放和最小内容尺寸配置，并支持文本元素 resize 后内容按宽度自动换行。
- 为 BDrawing 画布增加右键菜单，支持复制粘贴、组合/取消组合、层级调整和删除选区。
- 调整 BDrawing 右键菜单合并项展示逻辑，保留多选右键选区并恢复多元素粘贴。
- 修复 BDrawing 右键菜单层级操作被 Selecto 误判为空白框选后清空选区的问题。
- 修复 BDrawing 多选 Moveable 控制框右键时未打开右键菜单的问题。
- 调整画图左侧图层列表为倒序展示，并支持在侧栏直接复制和删除图层。
- 调整画图图层操作按钮为悬停显示的 BButton，并让侧栏复制元素插入到原图层上方后选中新副本。
- 为画图左侧图层列表增加拖拽排序，支持按侧栏视觉顺序调整画布图层层级。
- 为画图左侧图层列表增加组合图层展示，合并元素会显示为组合行并展开组内图层。
- 调整画图组合图层为可折叠树形展示，子图层复用普通图层行样式并仅保留缩进。
- 支持画图组合子元素作为设置面板编辑目标，并在视图区同时高亮组合选区与当前子元素。
- 调整画图组合子元素激活态，选中子元素时隐藏 Moveable 组内其他子目标默认线。
- 调整画图组合子元素激活高亮由 Moveable 控制层渲染，避免节点自身圆角和边框影响选中框。
- 将 BDrawing 新建和复制元素 ID 改为 8 位 nanoid，并按当前已有同类型标题生成 `矩形1`、`文本1` 等默认名称。
- 为画图右侧设置栏增加基于 BSection 的多选元素看板，展示类型、范围、组合状态并提供快捷操作。
- 修复画图组合拖拽预览中当前子元素 Moveable 控制框不跟随移动的问题。
- 修复画图取消组合或画布多选后侧栏与设置面板选中态残留的问题。
- 修复画图组合子元素选中时左侧图层子行未高亮的问题。
- 按交互要求移除画图组合图层折叠按钮与拖拽手柄的 aria 属性。
- 为画图图层拖拽增加手柄图标，并修复拖拽到列表第一项上方时无法置顶的问题。
- 新增 BDraggable 公共列表拖拽排序组件，并将画图图层拖拽迁移到插槽式公共组件。
- 修复 BDraggable 图层拖拽与 HeaderTabs 标签拖拽监听互相串扰的问题。
- 调整 BDraggable 由组件内部管理列表项容器和 ref，并改为通过 class 查找拖拽手柄，避免插槽类型错误与指示器占位。
- 修复 BDraggable 最近边数据未写入 drop target，导致图层无法拖拽到视觉第一项前方的问题。
- 调整 BDraggable 原生拖拽预览为临时克隆节点，并隐藏拖拽中图层选中边框，避免出现两条插入线的错觉。
- 将 BDraggable 插入指示器收敛为组件级单一绝对定位线，并在无命中目标时按指针位置兜底解析首位插入。
- 调整 BDraggable 插入指示线坐标为相邻列表项间距中点，避免有 gap 时贴边显示。
- 修复 HeaderTabs 标签拖拽最近边数据未写回，导致左半区投放无法稳定插入到目标前方的问题。
- 为 BDraggable 拖拽数据增加实例隔离，避免多个拖拽列表共存时互相接收拖拽事件。
- 优化 BDraggable 和画图图层排序的无效移动处理，顺序未变化时不再触发数据写入。
- 为 BDraggable 增加拖拽结束事件，并将 HeaderTabs 标签拖拽迁移到公共 BDraggable 组件。
- 调整 BDraggable 默认拖拽手柄为当前列表项容器，HeaderTabs 不再显式传入 handle-class。
- 修复 HeaderTabs 拖拽插入线贴近标签边缘的问题，将标签间距移动到 BDraggable item 容器以便居中计算。
- 调整 BDraggable 的 itemKey 默认值为 `id`，减少常见列表使用时的必填配置。
- 为 BDrawing 右键菜单增加视口边界定位，靠近窗口边缘打开时自动收进可视区域。
- 恢复画图页面默认设置面板的画板概览信息展示。
- 将画图侧栏工具创建改为 `useDragger` 公共自定义拖拽，支持无边框主题化文字预览和拖拽期间全局 `grabbing` 光标。
- 调整 `useDragger` 拖拽预览阴影为无偏移居中阴影，避免视觉下坠。
- 调整 `useDragger` 拖拽预览高度为 20px，并在拖拽时保留鼠标按下位置的相对握持偏移。
