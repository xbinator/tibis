# 2026-07-12

## Added

- 新增 LongCat（美团）AI 服务商，包含 LongCat-2.0（reasoning，1M 上下文）、LongCat Flash Chat 和 LongCat Flash Thinking 三个模型。
- 新增商汤科技（日日新 SenseNova）AI 服务商，包含 SenseNova U1 Fast（原生理解生成统一模型，256K 上下文）和 SenseNova 6.7 Flash-Lite（轻量多模态智能体）两个模型。

## Changed

- Anthropic 服务商新增 Claude Fable 5 模型（Mythos 级，1M 上下文，128K 输出，自适应思考）。
- OpenAI 服务商新增 GPT-5.6 系列三款模型：Sol（旗舰推理）、Terra（均衡型）、Luna（速度与成本优先），均支持 1.05M 上下文和 128K 输出。
- 小米服务商更新为 V2.5 系列：MiMo V2.5 Pro（纯文本/代码旗舰）+ MiMo V2.5（全模态 Agent），移除已下线的 V2 系列（V2 Pro / V2 Omni / V2 Flash）。
- 火山引擎服务商新增豆包 2.1 系列：Doubao Seed 2.1 Pro（旗舰智能体）和 Doubao Seed 2.1 Turbo（轻量高效），保留 2.0 系列向下兼容。
- 腾讯云混元 Hy3 正式发布（7 月 6 日），修正 Hy3/Hy3 Preview 为纯文本模型（不支持视觉/视频），描述补充 Agent 任务能力。
- 百度智能云 ERNIE 5.1 描述更新为“多模态理解与智能体应用”，反映其全模态能力定位。
- Skill 与 Widget 统一使用可配置冲突策略的原子目录安装事务和原生文件接口，移除重复安装 API 适配层，并增加 Windows 瞬时占用重试、失败回滚、阶段日志与原始错误提示。
- 目录安装事务增加崩溃恢复记录、主进程跨窗口目标锁与安装器级安全相对路径校验，避免中断替换丢失旧资源、并发覆盖和路径穿越。
- Skill、Widget、ZIP 与安装器统一复用共享路径规则，支持 Windows UNC 目录；路径状态查询保留权限错误，事务清理失败时保留恢复记录并记录恢复失败日志。
- Widget 创建流程收口到 `WidgetCreator.vue`，由组件闭环处理校验、目录安装、列表同步和编辑器打开；通用目录模块简化命名为 `directory.ts`。
- 运行日志页面增加主动刷新入口并适配行为测试，筛选控件统一为 28px 高度和 12px 字体，Skill 安装操作按钮文案统一为“确定”。
