# 2026-04-10

## Changed
- 优化 `BScrollbar` 和 `BTruncateText` 的尺寸监听方式，降低 `ResizeObserver loop completed with undelivered notifications` 警告出现概率。
- 将 TypeScript 调整为最新稳定版 `5.9.2`，移除对 TypeScript 6 兼容参数的依赖。
- 将 Electron 主进程准备脚本改为跨平台 Node 脚本，修复 Windows 下 `npm run electron:dev` 无法直接启动的问题。
- 隐藏 Electron 窗口标题栏：macOS 使用 `titleBarStyle: 'hidden'`，Windows/Linux 使用 `frame: false`。
- 将 Electron 主进程和 preload 脚本迁移到 ESM，移除 CommonJS 兼容 hack（`electron/package.json` 和 `prepare-electron-main.mjs`）。
- 修复 ESM 迁移后 `electron-store` 加密密钥不稳定导致无法读取存储数据的问题，改为将密钥持久化到独立文件。
- 修复 ESM 模式下 `__dirname` 未定义的问题，使用 `import.meta.url` 和 `fileURLToPath` 替代。

## Added
- 新增 `BTitleBar` 组件，为 Windows/Linux 提供自定义标题栏，包含最小化、最大化、关闭按钮。
- 新增窗口控制 IPC 接口：`window:minimize`、`window:maximize`、`window:close`、`window:isMaximized`。
