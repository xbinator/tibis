# 2026-05-25

## Added
- 新增 Tibis 工作区根目录提供器（`electron/main/modules/workspace/root.mts`），默认路径 `~/Tibis`，自动创建并 realpath 规范化
- 新增 IPC handler `workspace:get-root`，供渲染进程获取工作区根目录
- Native Bridge 新增 `getTibisWorkspaceRoot()` 方法，Electron 端调用 IPC，Web 端返回 null
- Shell 工具新增 Tibis 工作区回退逻辑：调用方未提供 workspaceRoot 时自动使用 `~/Tibis`
- 新增工作区根目录测试（4 个用例）和 Shell 工具回退测试（2 个用例）

## Changed
- Shell 安全检查放行 `/dev/null`（bash）和 `$null`/`Out-Null`（PowerShell）重定向目标，不再误判为工作区外写入
- Shell 工具描述和参数说明从"当前工作区/工作区根目录"改为"Tibis 工作区/Tibis 工作区目录"
- Shell 工具缺少 workspaceRoot 时的错误消息改为"无法初始化 Tibis 工作区目录"
- 确认描述文案改为"AI 请求在 Tibis 工作区内执行 Shell 命令"
