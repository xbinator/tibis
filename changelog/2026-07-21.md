# 2026-07-21

## Changed

- 全局将旧背景色属性文本统一改为 `background`，覆盖样式声明、过渡属性、相关测试断言与历史文档记录。
- `skills/developing-widget/SKILL.md` 与 `agents/openai.yaml` 改为中文说明，并强化 Widget 安装路径规则：安装或持久化时只允许写入 `~/.tibis/widgets/<widget-id>/widget.json`；同步收紧 `references/widget-format.md` 的安装位置说明，移除完整错误路径示例并明确 `<widget-id>` 是小组件标识而非 `widget.json.name` 显示名。
- `ServerCard.vue` 合并重复的错误信息展示：移除 `server-card__error` 独立错误块，状态行内的 `status.message` 统一通过 `server-card__status-message` 类承载，并在 `runtimeStatus === 'failed'` 时附加 `--failed` 修饰符以保留错误色（`var(--color-error)`），消除模板与样式中的重复定义。
- `ServerCard.vue` 模板类名统一接入 `createNamespace('', 'server-card')` 生成的 `bem` 函数：所有硬编码的 `class="server-card__xxx"` 改为 `:class="bem('xxx')"`，根节点使用 `:class="name"`；状态消息修饰符改为 `bem('status-message', { failed: ... })` 由 BEM 函数动态拼接。`<style>` 中仍保持完整类名选择器以符合项目规范（禁止 `&__xxx` 嵌套生成子类名）。
- `SlashCommandSelect.vue` 斜杠命令项标题展示内容由 `item.title` 改为 `item.trigger`，对应模板与样式中的类名 `slash-command-item-title` 同步重命名为 `slash-command-item-trigger`；`slash-command-select-scroll.test.ts` 中相关断言（`exists` 与文本期望值）同步更新。
- 调整上下文压缩中止逻辑：取消压缩时只收敛压缩消息状态，不再额外追加中断消息或新压缩片段。
- 修复 `bubble-part-tool-shell.test.ts` 中 "keeps the shell command text at the terminal default color" 用例：原测试用正则匹配独立的 `.bubble-part-tool__shell-command { ... }` 样式块并要求含 `display: flex`，但组件实际通过相邻兄弟选择器控制布局、命令文本颜色由终端区域继承保持默认色，导致匹配为空而断言失败。改为校验所有作用于 Shell 命令容器及其相邻布局的样式规则均不含 `color: var(--text-tertiary)`，贴合真实实现并保留弱化色回归防护。
- 移除 speech 语音功能：删除聊天输入语音录制/转写组件、语音设置页、Electron speech IPC/runtime、打包资源、开发脚本、发布 workflow 与当前语音文档，并新增静态测试防止入口残留。
