# 2026-07-19

## Changed

- AI 请求超时策略不再设置单步 `stepMs` 限制，避免长流式回答在持续输出时被 120 秒 step timeout 中断。
- 主进程工具等待用户确认时暂停工具执行超时，避免长时间未操作导致 `write_file` 等工具被误报为 60 秒执行超时。
- Runtime 等待用户确认时同步暂停任务级 300 秒执行时钟，并避免 SDK 层 total timeout 误计入人工等待。
- Shell 安全策略将小范围删除命令标记为需用户确认，并阻止递归强制、通配和工作区外删除。
- Widget 页面在最近记录缺少路径时会从已安装 Widget 列表兜底解析 `widget.json` 路径，避免从设置页打开后显示无数据。
- Widget 标签页标题对已安装 `widget.json` 显示目录 ID，避免顶部标签栏显示固定文件名。
- 默认 AI 服务商模型的 `type` 字段统一收敛为 `chat` / `image` / `video` 三类，`reasoning` / `code` / `vision` 全部映射至 `chat` 或 `image`，并移除模型编辑弹窗中的「多模态模型」选项，避免与设置页分类筛选项不匹配导致模型在列表中无法显示。
- `AIProviderModel.type` 收紧为 `AIModelType = 'chat' | 'image' | 'video'` 联合类型，编译期阻止非法的模型 type 值回归。
- 默认 AI 服务商内置模型更新：Anthropic `claude-sonnet-4-6` 升级为 `claude-sonnet-5`（2026-06-30 发布，主打代理能力），Moonshot `kimi-k2.6` 升级为 `kimi-k3`（2026-07-16 发布，2.8 万亿参数、1M 上下文、前端代码 WebDev Arena 全球第一）。
