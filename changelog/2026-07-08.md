# 2026-07-08

## Added
- 新增 `glob` / `grep` 主进程 AI 文件搜索工具设计与实现：`glob` 使用 Node 遍历和 `picomatch` 匹配文件路径，`grep` 使用系统 `grep -E` 搜索文件内容，并统一支持工作区边界校验、工作区外确认、`.git` 排除、结果截断、子进程超时取消和 stdout/stderr 上限。
- 新增 Widget 运行态生命周期开发文档，明确 session 实例身份、执行阶段、状态边界、脚本切换和事件方法调用约定。
- 新增热血红黑「Shonen」主题预设，提供漫画纸张浅色模式与高对比红黑暗色模式。
- 新增黑白线稿「Manga Ink」主题预设，提供纸张墨线浅色模式与反相线稿暗色模式。

## Changed
- 增强 `glob` / `grep` 文件搜索工具的部分成功能力：目录或单文件不可读时尽量返回已获得结果，并通过 `incomplete` 与 `warnings` 暴露非致命问题。
- 修复 `grep` 高命中搜索、路径含数字冒号、warning 过多和子进程超时清理问题：grep stdout 改为流式截断，parser 优先按候选文件路径解析，warning 返回增加上限，超时/取消后等待子进程退出并提供强制 kill 兜底。
- 降低 BWidget 小地图视口矩形拖拽灵敏度，减少拖拽时的视口跳动。
- 调整 Widget 页面左右侧面板为覆盖式布局，避免面板宽度变化挤压画布区域。
- 调整 Widget 页面右下角工具栏位置，使其避让覆盖式右侧设置面板。
- 调整 Widget 拖拽创建交互，拖拽期间临时隐藏左侧内容面板并在释放后恢复。
- 修复 BWidget Moveable 控制点 hover 与拖拽过程 cursor 不稳定的问题。
- 调整 Widget 显示期运行态为 session 模型，`onMounted`、方法事件与兼容交互脚本复用同一个实例，并保留 `open_widget` 的 onExecute 日志与 console 桥接。
- 修复 Widget 运行态 session 的超时释放、兼容交互实例复用和同组件脚本切换问题，避免 Worker 泄漏或按钮事件执行旧脚本。
- 修复 Widget 运行态 Worker payload 携带响应式对象时的 `DataCloneError`，脚本可见输入、输出、数据和方法参数会先转换为 JSON 快照。

## Removed
- 移除 Widget 默认脚本模板中的空 `confirm()` 方法，按钮事件需要由业务脚本显式定义。
