# 2026-07-08

## Added
- 新增 `glob` / `grep` 主进程 AI 文件搜索工具设计与实现：`glob` 使用 Node 遍历和 `picomatch` 匹配文件路径，`grep` 使用系统 `grep -E` 搜索文件内容，并统一支持工作区边界校验、工作区外确认、`.git` 排除、结果截断、子进程超时取消和 stdout/stderr 上限。

## Changed
- 增强 `glob` / `grep` 文件搜索工具的部分成功能力：目录或单文件不可读时尽量返回已获得结果，并通过 `incomplete` 与 `warnings` 暴露非致命问题。
- 修复 `grep` 高命中搜索、路径含数字冒号、warning 过多和子进程超时清理问题：grep stdout 改为流式截断，parser 优先按候选文件路径解析，warning 返回增加上限，超时/取消后等待子进程退出并提供强制 kill 兜底。
- 降低 BWidget 小地图视口矩形拖拽灵敏度，减少拖拽时的视口跳动。
- 调整 Widget 页面左右侧面板为覆盖式布局，避免面板宽度变化挤压画布区域。
- 调整 Widget 页面右下角工具栏位置，使其避让覆盖式右侧设置面板。
