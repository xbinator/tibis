# 2026-06-10

## Added
- 新增 `BDrawing` 技术路线与发展规划文档，明确自由画板采用 Moveable/Selecto、自有数据模型和未来脑图扩展方向。
- 新增 `BDrawing` 矩形、椭圆、菱形和文本工具，支持点击默认尺寸创建、拖拽自定义尺寸创建和创建预览。
- 新增 `BDrawing` Moveable 控制器适配层，支持选中元素后的拖拽、缩放、旋转结束提交。
- 新增 `BDrawing` Selecto 框选适配层，支持选择工具下框选替换选区和 Shift 追加选区。
- 新增 `BDrawing` 连接线工具，支持连接两个形状、随形状几何变化更新路径，并启用 Moveable 基础吸附。

## Changed
- 修复 Vite 构建中的动态导入无效告警，并关闭 Rolldown 插件耗时诊断提示。
- 细化 `BDrawing` 技术路线文档，补充数据模型、事件流、Moveable/Selecto 适配、迁移策略、测试矩阵和分阶段验收标准。
- 将 `BDrawing` 画板状态从 `nodes` 主模型迁移为 `elements` 主模型，并移除 Plait Core 依赖边界。
- 优化 `BDrawing` 创建体验：元素创建完成后自动回到选择工具，创建工具激活时隐藏选中控制器，并支持流程节点拖拽创建自定义尺寸。
- 简化 `BDrawing` 工具区，移除流程节点新增按钮、删除按钮和空画布占位文案。
- 修复 `BDrawing` 创建工具拖拽时 Selecto 框选层与创建预览框同时出现的问题。
- 修复 `BDrawing` 节点拖拽过程中没有实时位移反馈、仅在鼠标抬起后更新位置的问题。
- 优化 `BDrawing` 节点拖拽体验，未选中节点可在同一次按下操作中完成选中并进入拖拽。
- 修复 `BDrawing` 节点缩放过程中没有实时尺寸反馈的问题。
- 调整 `BDrawing` 节点直接拖拽实现，移除 Moveable 延迟接管同次按下事件导致的卡顿和松开后继续跟随问题。
- 修复 `BDrawing` 节点直接拖拽在鼠标松开时使用抬起坐标重新计算位置导致的跳动问题。
- 修复 `BDrawing` 节点直接拖拽结束后 Moveable 拖拽结束事件重复提交导致节点再次偏移的问题。
- 移除 `BDrawing` Moveable 旋转控制，仅保留拖拽和缩放交互。
- 优化 `BDrawing` Moveable 拖拽体验，已选中节点保留 Moveable 拖拽，未选中节点拖拽时延迟到鼠标抬起后再显示 Moveable。
- 修复 `BDrawing` 拖拽未选中节点时旧 Moveable 选框仍在拖拽开始阶段显示的问题。
- 修复 `BDrawing` Moveable 未传入元素吸附参考节点，导致两个元素之间不显示吸附线的问题。
- 优化 `BDrawing` Moveable 吸附体验，显式启用中心线吸附并避免吸附距离数字换行显示。
- 修复 `BDrawing` 节点直接拖拽位移未按画布实际渲染尺寸换算，导致鼠标移动距离大于节点移动距离的问题。
- 调整 `BDrawing` Moveable 吸附策略，多选时关闭 snappable，仅单选时启用吸附。
