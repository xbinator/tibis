# 2026-07-07

## Changed
- [BWidget/BChat] 重构 `open_widget` 运行时协议：工具调用阶段先执行可选 `onExecute` 并把执行状态写入 tool result，展示阶段再执行 `onMounted`；Widget 脚本上下文改为只读 `this.$input` / `this.$output`，模板变量、预览上下文和编辑器变量选择同步迁移到 `$input` / `$output`。
- [BChat] 收紧小组件消息职责：`this.$sendMessage` 仅表示发送消息请求，交互阶段消息仍由聊天层根据流式状态统一拦截；`onExecute` 内部的 `$sendMessage` 不向 Chat 泄漏。
- [BChat] 修复 `open_widget` 工具注册链路，统一使用 renderer 侧执行器运行 `onExecute`，并为 `onExecute` 阶段传入托管 `$http` 客户端。
- [BWidget/widgetRuntime] 把 `dataPatch` 概念重命名为 `patch`：
  - 文件 `dataPatch.ts` → `patch.ts`。
  - 类型 `WidgetRuntimeDataPatch` → `WidgetRuntimePatch`、`WidgetRuntimeDataPathSegment` → `WidgetRuntimePathSegment`、`WidgetRuntimeDataPatchableState` → `WidgetRuntimePatchableState`。
  - 函数 `applyWidgetRuntimeDataPatch` → `applyWidgetRuntimePatch`、`applyWidgetRuntimeDataPatches` → `applyWidgetRuntimePatches`、`applyWidgetRuntimeDataPatchesToState` → `applyWidgetRuntimePatchesToState`、`isWidgetRuntimeDataPatch` → `isWidgetRuntimePatch`、`isWidgetRuntimeDataPatchArray` → `isWidgetRuntimePatchArray`。
  - 回调 `onDataPatch` → `onPatch`，本地函数 `commitRuntimeDataPatches` → `commitRuntimePatches`。
  - 沙箱内部辅助 `__flushDataPatches` → `__flushPatches`、`__scheduleDataPatchFlush` → `__schedulePatchFlush`、`__recordDataPatch` → `__recordPatch`、`__flushScheduledDataPatchFlushes` → `__flushScheduledPatchFlushes`、`__createSetDataPatch` → `__createSetPatch`；执行态字段 `pendingDataPatches` → `pendingPatches`、`pendingDataPatchFlushes` → `pendingPatchFlushes`、`pendingDataPatchFlushScheduled` → `pendingPatchFlushScheduled`。
  - 沙箱宿主函数名 `SANDBOX_WIDGET_DATA_PATCH_HOST_FUNCTION_NAME` → `SANDBOX_WIDGET_PATCH_HOST_FUNCTION_NAME`，运行时字符串 `'__sandboxWidgetDataPatch'` → `'__sandboxWidgetPatch'`。
  - 同步更新 [BWidget/Runtime](src/components/BWidget/Runtime.vue) 的 import 与函数引用，以及 `test/components/BWidget/widget-runtime-patch.test.ts`（由 `widget-runtime-data-patch.test.ts` 改名）、`test/components/BWidget/widget-runtime-logger.test.ts`、`test/components/BChat/widget-runtime.test.ts` 三处测试；所有 JSDoc 中「data patch」措辞统一改为「patch」。
  - tsc + eslint 通过，BWidget / BChat 测试套件 76 个文件 569 个测试全部通过，无回归。

## Removed
- [BWidget] 移除 Widget 脚本历史 `mounted` / `unmounted` 生命周期、`this.$execute` API，以及 `open_widget` 展示 payload 中的 `kind: 'widget_display'` 字段。
