# 2026-06-06

## Added
- 新增默认布局 Header 动态工具栏注册能力，支持当前激活标签页按需显示编辑器工具。

## Changed
- 将 Markdown 与 Monaco 编辑器工具栏迁移到默认布局 Header 右侧动态渲染。
- Header 动态编辑器工具栏存在可见数据时，在右侧追加分割线以区分常驻按钮。
- WebView 页面最近记录仅在首次导航完成时写入，避免页面内链接切换反复刷新最近记录。
- 最近文件记录读取与系统快捷入口同步兼容历史缺失字段，避免欢迎页加载时报 `trim` 异常。
- macOS 关闭所有窗口后保持数据库连接，避免重新激活窗口读取聊天历史时报数据库未初始化。
- 第二实例与系统快捷入口动作会等待主进程完成存储、数据库和 IPC 初始化后再派发，避免窗口早于资源启动。
- 系统“打开方式”传入的 Markdown 文件改为主进程队列 + 渲染进程主动消费，支持冷启动与 Windows/Linux 命令行路径打开编辑器。
- MCP OAuth 授权流程改为使用主进程返回的授权 URL 主动打开浏览器，避免依赖未注册监听的事件。

## Removed
- 移除 Markdown 与 Monaco 编辑器内部固定工具栏占位。

## Features
- 支持 Markdown 大纲、编辑模式、文件操作、页宽设置以及 Monaco JSON 视图、自动换行在 Header 中跟随激活标签页切换。
