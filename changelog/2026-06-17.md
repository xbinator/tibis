# 2026-06-17

## Added
- 记忆内容展示组件（`src/views/settings/tools/memory/components/MemoryContent.vue`）新增鼠标悬停时右上角显示重置按钮，点击后通过 `Modal.delete` 弹出确认框，确认后调用 `memoryStore.clearAll()` 清空全部记忆。确认框正文分两段：主文案 + 小字灰色提示（说明记忆开关与重置的关系）。
- BMessage Markdown 支持代码块语法高亮与一键复制、KaTeX 数学公式渲染、Mermaid 图表预览。

## Changed
- `src/utils/modal.tsx` 的 `Modal.confirm` / `Modal.delete` 的 `content` 参数类型由 `string` 扩展为 `string | VNode`，支持富文本/分段样式内容。`Modal.delete` 的 `options` 新增 `width` 字段支持自定义弹窗宽度。
- BEditor Rich 模式 Markdown 导入支持 `<u>`、`<mark>`、`<br>`、`<kbd>`、`<abbr>`、`<small>`、`<sub>`、`<sup>` 等安全行内 HTML 标签的富文本语义渲染。
- BEditor Rich 模式块级数学公式支持预览 / 编辑切换，可直接修改 LaTeX 内容并同步回 Markdown。
- BEditor Rich 模式代码块在 Mermaid 预览可见时支持复制渲染后的 PNG 图片，普通代码视图仍复制源码文本。
- BEditor Rich 模式代码块复制源码与 Mermaid 预览图时统一使用“复制成功”提示。
- BEditor Rich 模式代码块复制逻辑统一使用项目内部剪贴板 Hook，源码复制保留原始空白。
- 新增图片工具 `src/utils/image.ts` 封装图片下载与 SVG 元素转 PNG 能力，剪贴板 Hook 负责复用该能力复制图片，BEditor 代码块不再承载图片转换细节。
- 图片工具 `src/utils/image.ts` 采用 `[Error] | [undefined, T]` 结果模式返回失败信息，不再向调用方抛出业务错误。
- 剪贴板 Hook 的图片复制 API 收敛为统一 `copyImage(source, options)`，支持 URL、SVG 元素与图片二进制来源。
- BChat 回到底部按钮支持鼠标悬停时暂停自动隐藏，鼠标移出后重新按空闲延迟倒计时隐藏。
- BChat assistant 草稿持久化改为串行收尾，最终消息落库前等待已启动的草稿写入，避免正常完成后被旧草稿覆盖并误展示硬中断恢复提示。
- BChat 硬中断恢复逻辑保留等待用户选择的 assistant 暂停态，避免刷新后把选择题误标记为意外中断。
- BDrawing 空画板完成首次视口初始化后，用户新增第一个节点时不再自动适配视口。
- BDrawing 连接线工具支持从画布空白区域拖拽创建独立线条。
- BDrawing 选中独立线条后支持拖拽端点调整线条位置。
- BDrawing 拖拽选中线条端点时不再触发 Selecto 框选。
- BDrawing 拖拽选中线条端点时，线条路径会实时跟随鼠标移动。
- BDrawing 拖拽选中线条端点到节点上方时会显示节点锚点并在释放后关联节点。
- BDrawing 选中线条后按下端点拖拽时，所有节点会显示可连接锚点提示。
- BDrawing 拖拽已选线条端点时会穿透端点覆盖层命中下方节点，确保释放后正常关联节点。
- BDrawing 节点缩放控制点仅保留左上、右上、左下、右下四个角，移除上下左右边中点控制。
- BDrawing 文本节点编辑态下开始拖拽时会先提交并退出编辑态，再继续移动节点。
- BDrawing 拖拽已选线条端点时隐藏被拖拽的旧端点圆圈，避免出现原点残影。
- BDrawing 连接线 hover 锚点渲染改为先收窄端点类型，修复 Vue TS 插件的联合类型报错。
- BDrawing 连接线折线路由会在端点锚点反向时按起点和终点节点边界计算外侧轨道，同侧锚点优先走共享外侧轨道，并覆盖混合锚点组合，避免线条穿过自身节点或绕远路。
- 服务商 API 配置页连通性检查区域支持容器响应式布局，窄容器下自动改为纵向排列。
- HeaderTabs 拖拽指引条按鼠标首尾边界优先于 sticky target 计算，修复拖到第一个标签前方时指引条偶发不显示的问题。
