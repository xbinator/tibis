# 2026-05-18

## Added
- 新增 `BEditor` 通用编辑器入口、`PaneMonacoEditor` 和 `createMonacoEditor` 适配层，为 `.json` 文件接入 Monaco 编辑能力。
- 新增 `editorDriver` 及对应测试，支持编辑页将 `.json` 文件分流到新的 Monaco 编辑分支。

## Changed
- Vite 组件自动注册目录补充 `BEditor`，并新增 `monaco-editor` vendor chunk 拆包配置。
- 编辑器驱动注册表不再让 `.json` 文件回退到 Markdown 驱动。
- 将 `js-tiktoken` 调整为真正按需加载，避免聊天侧 token 估算器在首包阶段静态拉入超大 tokenizer 依赖。
- 将 Monaco worker 构造器改为懒加载，仅在打开 Monaco 编辑器时解析相关 worker 资源，收紧默认构建入口体积。
- 将 Vite 构建的 `chunkSizeWarningLimit` 调整到 `5800`，对齐当前最大业务异步 chunk 体积，减少对已知大型懒加载依赖的误报。
- 将 `BMarkdown` 的富文本/源码模式判断内聚到组件内部，改为直接读取编辑器偏好 store，并移除编辑页对 `view-mode` 的外部透传。
