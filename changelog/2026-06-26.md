# 2026-06-26

## Changed
- 调整画图设置面板选中元素编辑链路，使用 v-model 转发选中元素引用更新并保留画板选区。
- 简化画图设计设置字段绑定，移除路径式代理对象并直接读取和更新选中元素字段。
- 调整画图设计设置名称输入，使用 `v-model:value` 直接绑定选中元素自定义标题。
- 重构 BDrawing 形状元素数据模型，使用必填样式、自定义元数据、注册名称与图标、默认跟随标签的自定义标题，并移除通用文本字段。
- 为 BDrawing 元素注册配置新增内容尺寸策略，文本元素按标题内容宽高渲染并同步选中框尺寸。
- 清理 BDrawing 旧版流程图工具残留，移除 `process` 专用 ID 分支和硬编码创建工具列表。
- 修正 BDrawing 内容尺寸元素的左上角创建锚点，并清理旧版文本自动换行撑高工具。
- 修正 BDrawing 文本元素选中时 Moveable 间距提示重复显示的问题，并保留正常距离提示。
- 为 BDrawing 设计设置面板新增边框、圆角和内边距控件，支持四边与四角定向编辑。
- 重构 BDrawing 对外模型，改用 `v-model:value` 绑定画板数据，并通过 `v-model:select` 暴露当前可编辑目标。
- 抽离 BMessage 图片行内节点为独立 ImageNode 组件，收拢图片预览、复制和加载错误处理逻辑。
- 迁移设置页布局组件到 `src/views/settings/_components`，使用 `SettingsPage` / `SettingsSection` 私有组件替代全局 `BSettings*` 组件。
- 收敛 MoveableLayer 吸附能力，多选或选中文本时关闭吸附辅助线和控制间距，单选矩形时保留。
- 合并 DesignSetter 圆角到边框区块，移除未使用的颜色选择器偏移常量。
- 移除 DrawingNode 选中轮廓线样式和 ControlPanel toggle 激活背景色，选中状态由 Moveable 控制框表达。
- BSection Item 前缀图标添加 `flex-shrink: 0` 和 `font-size: 12px` 防止布局压缩。
- InspectorPanel 移除标签固定字号覆盖，字号由 BSection Item 全局统一控制。

## Removed
- 移除 `interaction-constants.test.ts` 常量测试，替换为 MoveableLayer 组件级测试。
