# 2026-07-09

## Removed
- 移除 `src/components/BChat/components/ConversationView.vue` 空状态中的装饰卡片插画（`conversation-view__art` 及其 `.conversation-view__card` 前后卡片与 `lucide:messages-square` 图标），并删除对应 CSS；空状态保留「开始对话」标题与提示文字，`BIcon` 仍在底部回到底部箭头处使用。

## Changed
- 禁止 BWidget 空白画布区域打开自定义右键菜单，保留元素右键菜单行为。
- 为 Widget 数据源侧栏新增入参 / 出参 schema 切换，出参 schema 写入 `outputSchema` 并用于 `$output` 类型提示和 Widget 工具契约。
- 移除 BWidget 画布主容器的直接边框，由外层页面承载画布边界。
- 新增通用 `BMonaco` 弹窗编辑器，并将 Widget Schema JSON 编辑迁移为侧栏解析与通用文本弹窗；打开时当前 schema 缺失则使用默认 schema 初始化草稿。
- 简化 Widget Schema JSON 编辑弹窗调用，直接绑定当前 schema，弹窗标题统一为「编辑」。
- 简化 Widget Schema 编辑模式与说明抽屉的字段选择逻辑，分段值和 SchemaHelp 内容映射直接使用 `inputSchema` / `outputSchema` 字段名，并使用 `lodash-es` 判断 JSON 编辑器保存值是否为普通对象。
- 移除 Widget Schema 分段切换中的冗余字段名 guard，切换函数直接接收 `inputSchema` / `outputSchema` 类型值。
- 移除旧的 Widget schema kind 分类，Widget schema 归一化统一使用对象 schema 默认值，不再传入 `data` 类型参数。
- 为通用 `BMonaco` 弹窗增加保存前校验能力，避免 Widget Schema JSON 编辑输入非对象 schema 时静默清空旧配置。
- 新增 BWidget schema 形状校验工具，并在 WidgetData 归一化与 Widget Schema JSON 编辑保存前复用对应判断，缺少顶层 `properties` 或 `required` 形状非法时阻止保存，避免无效 schema 覆盖旧配置。
- 移除 Widget 入参 Schema JSON 编辑弹窗中的 `kind` 配置链路，编辑器仅接收当前 schema 并使用统一对象 schema 默认值。
- 为 `BButton` 新增 `soft` 类型，提供主题色文字和浅主题色背景，并补充危险状态样式。
- 将 BChat 消息回退的二次确认从 `Modal.confirm` 弹窗改为 `MessageBubble` 内联确认条：回退按钮点击后展开「回退将删除该消息之后的所有内容，不可撤销。」提示与「取消 / 确认回退」按钮，确认后才向上 emit `rollback`；同步移除 `index.vue` 中 `handleRollback` 的 Modal 调用、`afterCount` 计数与 `Modal` 导入，确认文案不再展示消息数量。
- 为 BChat 回退内联确认条新增标题「确定要回退至此回答重新发起？」并以 `box-shadow: 0 0 0 1px var(--border-primary)` 描边，避免影响圆角。
- 修复 BWidget 已选中单个元素直接拖拽时选中控制层临时消失的问题，仅在新选中拖拽或组合拖拽时隐藏 Moveable 控制层。
- 修复 BWidget 节点拖拽松手后位置跳动的问题：已选中单节点移动交由 Moveable 提交，直接拖拽结束时使用 pointerup 最终坐标提交。
- 修复 BEditor 段落内 Markdown 图片多余外边距：为 `p .b-markdown-image` 设置 `margin: 0`，避免图文混排时图片上下留白。
- 为 BWidget 页面新增运行态展示宽高配置，运行态按 `metadata.width` / `metadata.height` 等比展示并在宿主宽度不足时缩小。
- 为 BWidget 页面尺寸配置区块补充提示，说明留空自适应、容器不足时缩小以及容器更大时保持配置尺寸。
- 为 `WidgetData` 增加顶层 metadata 泛型参数，默认仍使用 `WidgetMetadata`。
- 重命名 BWidget 运行态布局 hook：`useRuntimeDisplayLayout` → `useRuntimeLayout`（文件 `hooks/useRuntimeDisplayLayout.ts` → `hooks/useRuntimeLayout.ts`，内部 `UseRuntimeDisplayLayoutOptions`/`UseRuntimeDisplayLayoutReturn` 同步改为 `UseRuntimeLayoutOptions`/`UseRuntimeLayoutReturn`），并同步更新 `Runtime.vue` 的导入与调用。
- 重命名小组件脚本类型提示工具：文件 `src/views/widget/constants/methodScriptExtraLib.ts` → `methodScriptTypes.ts`，导出函数 `createWidgetMethodScriptExtraLibContent` → `createWidgetMethodScriptTypes`，并同步更新 `CodeEditor.vue` 的导入与调用，以及相关 spec / plan 文档与历史 changelog 中的引用。
- 为 BWidget 新增按钮元素，支持工具栏创建、画布渲染和右侧按钮文字配置，并修复文本元素清空最大行数后仍启用截断布局的问题。
- 增强 BWidget 按钮元素设置，移除尺寸配置，新增 `BTextSelect` 变量输入切换和 `BTextMethod` 多函数动作配置，变量输入展示时隐藏外层 `{{ }}`，点击动作统一保存为 `actions[].method` / `actions[].args`。
- 为 `BTextInput` 新增 `replaceEntireValue` 布尔配置，支持变量选择时按光标插入或覆盖整个输入值。
- 优化 `BTextMethod` 动作设置弹窗布局，增加左右面板与参数项边框层次。
- 简化 `BTextMethod` 动作设置布局，改为外层 actions 列表展示并支持直接编辑、删除动作。
- 调整 `BTextMethod` 动作设置弹窗为左右双边框面板布局，并压缩外层 actions 列表高度。
- 为 `BTextMethod` 动作设置弹窗增加稳定默认面板高度，并在切换函数时保留已有参数值。
- 移除 `BTextMethod` 动作设置弹窗的 section meta 与参数底部操作区，将「添加参数」移动到参数标题区域。
- 固定 `BTextMethod` 触发按钮文案为 `placeholder`，不再随已配置动作数量或函数名变化。
- 调整 `BTextMethod` 已配置动作行的操作区交互，参考 Widget 图层列表改为 hover/focus 显示的轻量图标按钮。
- 为 `BTextMethod` 已配置动作列表接入 `BDraggable` 拖拽排序，将参数删除按钮改为 `BButton`，并移除 action 编辑 / 删除专用类名。
- 简化 `BTextMethod` 弹窗草稿状态为单个 `draftAction`，移除 `data-action` 测试标识属性。
- 将 `BTextMethod` 弹窗临时动作状态重命名为 `editingAction`，并移除 `activeAction` 计算层，保留取消 / 确认的编辑副本语义。
- 修复 `BTextInput` 变量下拉未按最近滚动容器可视空间选择展开方向的问题，避免靠近容器底部时被裁切隐藏。
- 优化 `BTextSelect` 静态选择态的变量值处理，变量模板不再注入为临时选项，切回选择态时保持空选中。
- 新增 `useElementMethods` hook 统一生成 Widget 元素动作方法候选，并移除旧的 `utils/widgetMethods.ts` 工具入口。
- 修复 `BTextSelect` 外部模型值从变量模板切回静态值时未退出变量输入态的问题，并补充 SenseNova provider 图标映射。
- 优化 BWidget 按钮元素运行态视图代码结构，统一按钮元数据读取、布尔状态解析与点击动作执行入口。
