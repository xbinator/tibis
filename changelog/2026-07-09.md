# 2026-07-09

## Removed
- 移除 `src/components/BChat/components/ConversationView.vue` 空状态中的装饰卡片插画（`conversation-view__art` 及其 `.conversation-view__card` 前后卡片与 `lucide:messages-square` 图标），并删除对应 CSS；空状态保留「开始对话」标题与提示文字，`BIcon` 仍在底部回到底部箭头处使用。

## Changed
- 将 BChat 消息回退的二次确认从 `Modal.confirm` 弹窗改为 `MessageBubble` 内联确认条：回退按钮点击后展开「回退将删除该消息之后的所有内容，不可撤销。」提示与「取消 / 确认回退」按钮，确认后才向上 emit `rollback`；同步移除 `index.vue` 中 `handleRollback` 的 Modal 调用、`afterCount` 计数与 `Modal` 导入，确认文案不再展示消息数量。
- 为 BChat 回退内联确认条新增标题「确定要回退至此回答重新发起？」并以 `box-shadow: 0 0 0 1px var(--border-primary)` 描边，避免影响圆角。
- 修复 BWidget 已选中单个元素直接拖拽时选中控制层临时消失的问题，仅在新选中拖拽或组合拖拽时隐藏 Moveable 控制层。
- 修复 BWidget 节点拖拽松手后位置跳动的问题：已选中单节点移动交由 Moveable 提交，直接拖拽结束时使用 pointerup 最终坐标提交。
- 修复 BEditor 段落内 Markdown 图片多余外边距：为 `p .b-markdown-image` 设置 `margin: 0`，避免图文混排时图片上下留白。
