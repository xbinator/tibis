# 2026-06-08

## Added
- 聊天工具结果中的 AI 创建、写入、修改文件摘要支持点击文件 chip 直接打开对应文件。
- 标题栏右侧新增 GitHub Release 更新提示，支持打开下载页并永久关闭提示。
- 帮助菜单新增“检查更新”入口，支持主动检查并打开新版本下载页。
- Electron 系统帮助菜单新增“检查更新”入口，并复用主动检查更新流程。

## Changed
- 修复欢迎页拖拽本地文件时无法通过 Electron 路径打开的问题，统一复用文件打开入口。
- 修复 BSearchRecent 输入框 `@keydown.enter` 与 `@keydown.esc` 触发 `onKeydown` 重复属性的类型错误，改为统一由 `handleKeydown` 分发处理。
- 快捷键帮助面板文件操作分组新增“打开最近文件”条目，复用 `EditorShortcuts.FILE_RECENT` 常量。
