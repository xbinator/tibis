# 2026-06-13

## Changed
- 清理 BDrawing 连线渲染中的测试专用标记，改用组件自身 class 支撑拖拽预览和测试断言。
- BDrawing 连线工具激活时显示样式面板，并将草稿颜色、箭头和曲线配置应用到新创建的连接线。
- 聊天侧边栏流式 assistant 消息支持草稿持久化，并在硬中断后恢复为已中断的半截回复。
- BDrawing 支持在画板聚焦时通过 Ctrl+A / Cmd+A 全选全部绘图元素。
- BDrawing 文本工具创建元素后支持就地输入并提交文本内容。
- BDrawing 普通形状创建后不再默认显示中心文字，仅文本工具保留可输入文本。
- BDrawing 文本元素改为按内容自适应尺寸，默认透明背景，并支持文字颜色、字号和对齐设置。
- 修复 BDrawing 文本创建时编辑框背景与选中底框可见、空行换行渲染不稳定的问题。
- BDrawing 文本工具改为仅点击创建，并支持双击已有文本进入二次编辑。
- BDrawing 文本节点选中后禁止通过 Moveable 修改尺寸，保持尺寸由文本内容自适应。
- BDrawing 文本编辑框移除背景、边框和阴影，避免编辑态影响文字颜色观感。
- BDrawing 文本编辑框改为基于视口定位，避免宽度被画板父容器裁剪。
- BDrawing 统一文本标注交互：text 工具使用输入光标，点击已有元素创建新文本，双击已有文本优先编辑，并在编辑态隐藏原文字和选中框。
- BDrawing 文本编辑框关闭浏览器软换行，避免连续输入时内容自动折行并跳回首行。
- BDrawing 文本编辑器由 textarea 改为 contenteditable div，并在粘贴时仅保留纯文本。
- BDrawing 文本尺寸优先使用 canvas 实测文字宽度，并改进无 canvas 时的中英文宽度估算。
- BDrawing 文本编辑器输入增长时自动调整视口，并同步编辑态与展示态的字体、行高和内边距，减少切换跳动。
- BDrawing 文本展示态改为按编辑器相同的顶部内边距逐行排布，进一步减少编辑/展示切换位移。
