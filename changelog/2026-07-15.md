# 2026-07-15

## Changed

- 移除小组件设置页的重复 Store 初始化，统一由默认布局负责资源扫描与目录监听。
- 将 Skill 与 Widget 资源初始化 Hook 迁移至默认布局目录，明确其应用布局级职责。
- 将 Skill 与 Widget 初始化调整为仅扫描资源目录，并由 Store 在列表、详情、编辑器、聊天和工具首次使用时懒加载及缓存入口内容。
- 新增资源目录级监听与共享请求机制，合并并发首次读取，并在应用内保存资源后同步更新 Store 缓存。
- 统一使用 `getSkill`、`getWidget`、`getSkills` 与 `getWidgets` 从 Store 获取资源，由 Store 通过 `asyncTo` 吸收读取异常并移除旧 Skill 文件级监听链。
