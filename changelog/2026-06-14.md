# 2026-06-14

## Added
- BDrawing 支持双击普通形状编辑节点文字，双击连接线编辑标签文字。
- BDrawing 普通形状文字支持水平和垂直对齐控制，输入超出形状宽度时自动换行。

## Changed
- 拆分 BDrawing 入口组件的模型同步、视口尺寸同步和文本编辑逻辑到独立 hooks，补充对应单元测试。
- 移除 BDrawing components 目录本地组件文件名和组件引用中的 Drawing 前缀。
- 按业务域拆分 BDrawing 常量目录，集中管理画板、视口、文本、交互、小地图、样式和 DOM 常量。
- 移除 BDrawing 常量统一导出入口，改为按业务常量文件直接引入。
- 调整 BDrawing 样式面板层级控制位置，将其移至面板底部。
- 调整 BDrawing 普通形状文字编辑框定位，使编辑位置跟随文本垂直对齐方式。
