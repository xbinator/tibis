# 2026-07-06

## Changed
- `src/views/widget/constants/methodScriptExtraLib.ts`：`RequestResponse` 改造为泛型接口 `RequestResponse<TData = unknown>`，`data` 字段类型由 `unknown` 改为 `TData`。
- `src/views/widget/constants/methodScriptExtraLib.ts`：`WidgetHttpClient` 的 `get` / `post` / `put` / `patch` / `delete` 方法增加泛型参数 `<TData = unknown>`，返回类型对应调整为 `Promise<RequestResponse<TData>>`，缺省泛型保持原 `unknown` 行为，向后兼容。

## Features
- Widget 脚本编辑器 HTTP 客户端支持通过泛型显式声明响应体类型，例如 `this.$http.get<WeatherResult>(url)` 可在 Monaco 中获得 `data: WeatherResult` 的类型提示。
