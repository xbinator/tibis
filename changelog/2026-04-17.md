# 2026-04-17

## Changed

- 调整 `BPromptEditor` 的默认回车行为为直接发送，使用 `Ctrl+Enter` 或 `Cmd+Enter` 插入换行，并移除回车调试日志输出。
- 完善 `BChat` 消息重新生成流程，新增 `setSessionMessages` 用于同步会话消息持久化状态。
- `BSearchRecent` 移除文件时同步移除对应的标签页。
- 为 `BChatSidebar` 增加聊天空状态缺省图，未开始对话时展示引导插画和说明文字。
- 优化 `BEditor` 的 Markdown round-trip 行为，在富文本模式未发生语义变更时尽量保留用户原始输入中的 HTML 注释和引用式链接写法，并让 HTML 注释、引用式链接及其定义在富文本视图中以可编辑文本显示。
- 为 `BEditor` 富文本模式补充 `Ctrl`/`Cmd` + 点击链接时使用系统浏览器打开外链的能力。
