# 2026-06-29

## Added
- 添加 BMonaco 按行选区等待异步初始化完成的回归测试。
- 添加 Monaco 浅色主题选区透明度回归测试。
- 添加 Monaco 查找高亮颜色映射与主题重新注册回归测试。
- 添加 BMonaco 原生选区样式覆盖回归测试。
- 添加 BMonaco 关闭同词 selection 高亮的回归测试。
- `BColorPicker` 无默认插槽时自动渲染 `AInput` 输入框，支持直接输入/显示颜色值。
- `BColorPicker` 新增 `placeholder`、`allowClear`、`readonly`、`disabled`、`bordered` 输入框相关属性。

## Changed
- 修复 BMonaco 在 Monaco 初始化尚未完成时触发行范围选区会直接失败的问题。
- 调浅浅色主题下 Monaco 选区背景透明度，避免高饱和主题压暗语法高亮。
- 修复 Monaco 查找命中继续使用内置红橙默认高亮色的问题，并允许同名主题重新注册以刷新颜色。
- 修复 BMonaco 原生选区继续使用 Monaco 内部旧颜色变量导致高亮过重的问题。
- 关闭 BMonaco 的 Monaco 同词 selection 高亮，避免选中一个字段时其它相同字段被误标为选区。
- `BColorPicker` 输入框失焦或按回车时校验并应用颜色，面板选色与预设选色时同步更新输入框显示值。
- 将画图页面默认入参/出参 schema 调整为查天气示例，并在编辑器清空保存时按当前 schema 类型恢复对应默认值。
- 统一画图页面 Schema 类型定义，并在 Schema 说明抽屉关闭后重置示例展开状态。
