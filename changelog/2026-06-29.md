# 2026-06-29

## Added
- 添加 BMonaco 按行选区等待异步初始化完成的回归测试。
- 添加 Monaco 浅色主题选区透明度回归测试。
- 添加 Monaco 查找高亮颜色映射与主题重新注册回归测试。
- 添加 BMonaco 原生选区样式覆盖回归测试。
- 添加 BMonaco 关闭同词 selection 高亮的回归测试。
- `BColorPicker` 无默认插槽时自动渲染 `AInput` 输入框，支持直接输入/显示颜色值。
- `BColorPicker` 新增 `placeholder`、`allowClear`、`readonly`、`disabled`、`bordered` 输入框相关属性。
- 添加 BDrawing 文本元素动态绑定解析与设计期预览上下文测试。
- 添加 BPromptEditor 默认变量 value Chip 渲染回归测试。

## Changed
- 修复 BMonaco 在 Monaco 初始化尚未完成时触发行范围选区会直接失败的问题。
- 调浅浅色主题下 Monaco 选区背景透明度，避免高饱和主题压暗语法高亮。
- 修复 Monaco 查找命中继续使用内置红橙默认高亮色的问题，并允许同名主题重新注册以刷新颜色。
- 修复 BMonaco 原生选区继续使用 Monaco 内部旧颜色变量导致高亮过重的问题。
- 关闭 BMonaco 的 Monaco 同词 selection 高亮，避免选中一个字段时其它相同字段被误标为选区。
- `BColorPicker` 输入框失焦或按回车时校验并应用颜色，面板选色与预设选色时同步更新输入框显示值。
- 将画图页面默认入参/出参 schema 调整为查天气示例，并在编辑器清空保存时按当前 schema 类型恢复对应默认值。
- 统一画图页面 Schema 类型定义，并在 Schema 说明抽屉关闭后重置示例展开状态。
- BDrawing 文本元素支持通过 `{{ input.x }}`、`{{ state.y }}` 读取渲染上下文，并使用一个 `BPromptEditor` 同时编辑静态文本和动态模板。
- 画图页面设置中提供 `input` / `state` 设计期预览数据，用于验证文本模板渲染效果。
- 修复 `BPromptEditor` 变量菜单在窄面板里固定高度导致变量名、路径和说明重叠的问题。
- 将 BDrawing 文本元素的模板字段读写和变量候选生成抽离为 `useElementTemplate`、`useElementVariables` hooks，降低新增元素 Setter 的接入成本。
- 将 BDrawing 画布渲染上下文抽离为 `useRenderContext` provide/inject hook，移除 `DrawingCanvas` 与 `DrawingNode` 的中间层属性转发。
- 修复 BDrawing 变量候选对 `wind-speed`、`temperature.celsius` 等非标识符字段生成不可解析路径的问题。
- BDrawing 文本元素按解析后的动态内容测量渲染尺寸，并移除 Text 内容对旧 title / bindings 数据的兼容逻辑。
- 收窄 BDrawing 模板字段 hook 调用面，默认读取 `metadata.content`，并将文本元素动态内容测量封装为统一 renderSize 工具。
- 移除 BDrawing 外部 `renderContext` 入参，改为从画布 `metadata.previewContext` 读取预览上下文；模板 hook 改名为 `useElementTemplate` 并要求显式传入字段名。
- 优化 BDrawing 变量候选数据，避免变量路径同时出现在描述和插入值中导致菜单重复显示。
- `BPromptEditor` 默认根据变量选项将 `{{...}}` 渲染为显示变量 value 的 Chip，并保留自定义 Chip 解析器优先级。
- 修复画图页面动态预览 JSON 错误提示在预览上下文回填后残留的问题，并移除 PageSetter 冗余 `metadata` 入参。

## Removed
- 移除 BDrawing 文本元素专属的展示内容读取函数，统一通过元素展示内容 hook 和模板字段解析工具读取动态内容。
- 移除 BDrawing 旧版 `metadata.bindings` 读取类型与工具出口，统一使用模板字段内联 `{{ input.x }}` 绑定。
