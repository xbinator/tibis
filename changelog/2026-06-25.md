# 2026-06-25

## Changed
- 调整 `BCommandPanel` 最近记录搜索：文件记录不再匹配隐藏的 `content` 字段，仅按标题、文件名、扩展名和路径搜索。
- 移除 BChat 与 Drawing 的 AI 工具交互：聊天不再暴露画板工具、不再处理画板 bridge 请求，画板页不再注册 AI Drawing 上下文。
- 从共享工具 registry 和主进程工具分发中移除 Drawing 工具定义，避免 `create_drawing`、`read_current_drawing` 与 `apply_drawing_operations` 继续作为运行时工具存在。
- 将共享 AI 工具 registry 拆分为 PascalCase Tool 目录，并改用 `shared/ai/tools/index.ts` 作为统一聚合入口。
- 更新 AI 工具开发指南，补充 PascalCase Tool 目录、`shared/ai/tools/index.ts` 聚合入口和新增工具流程。
- 为画图页面新增基础三栏布局，左侧提供工具与图层入口，右侧展示画板基础设置信息。
- 将画图页面左侧栏和右侧设置栏抽离为 `src/views/drawing/components` 下的独立组件。
- 优化画图页面左侧栏面板结构，复用组件与图层面板标题区域。
- 调整画图页面左侧栏为左右布局，使用左侧图标页签栏承载工具与图层切换。
- 将画图页面左侧栏页签按钮改用 `BButton` 组件，统一按钮实现。
- 新增 BDrawing 元素注册目录，先注册矩形与文本元素，并让画图侧边栏组件区从注册表读取。
- 移除 BDrawing 连接线渲染、交互和底层数据类型链路，并删除浮动样式面板组件。
- 移除 BDrawing 创建预览渲染组件，并支持从画图侧边栏拖拽元素到画布创建。
- 将画图拖拽放置入口移动到页面画布容器，由 BDrawing 暴露方法完成元素创建。
- 优化画图侧边栏工具拖拽预览，保持拖拽状态下的圆角样式。
- 修复 BDrawing 缩放后选中元素拖拽位移被二次缩放，导致拖拽过程不跟随鼠标的问题。
- 移除画图侧边栏图层列表中的类型副标题展示。
- 将画图元素拖拽数据工具从 BDrawing 目录移动到画图页面工具目录，避免 BDrawing 组件处理拖拽协议。
- 将 BDrawing 主画布节点渲染从 SVG 切换为 HTML 节点，并通过元素注册表的 `View.vue` 渲染节点内容。
- 收窄 BDrawing 元素 schema 字段，仅保留 `name`、`label` 和 `icon` 展示配置，View/Setter 与创建适配改由元素注册入口维护。
- 修复 BDrawing 点击节点后 Moveable 选择框与节点错位的问题，将 Moveable 控制层改为画布绝对覆盖层。
- 修复 BDrawing Moveable 控制层与画布缩放平移坐标空间不一致，改为根层控制框配合准确定位，避免点击节点后节点消失。
- 简化 BDrawing 元素创建入口，移除注册名到形状类型的重复映射，直接使用 schema 的 `name` 创建元素。
- 修复 BDrawing 创建文本节点后直接进入编辑态导致节点看似未创建的问题，创建后直接展示节点内容。
- 移除 BDrawing 节点双击文本编辑链路和文本编辑覆盖层。
- 修复 BDrawing 多选节点拖拽时误用 Moveable 绝对 translate 作为增量导致节点跳动的问题。
- 修复 BDrawing 视口变更不会同步到外部 `drawingData` 的问题，确保右侧设置与保存数据使用最新视口。
- 修复 BDrawing 加载已有自动生成 ID 后继续创建元素可能产生重复 ID 的问题。
- 清理 BDrawing 已移除连接线、SVG 画布、样式面板和文本编辑器链路的旧测试，改为覆盖当前 HTML 节点渲染与注册元素创建能力。
- 移除 BDrawing 数据模型中的 `edges` 旧版占位字段，新建与保存的 `.tibis` 画图数据不再写入该字段。
- 清理 BDrawing 类型文件中未被外部使用的中转类型，改为在实际数据结构中直接声明字段。
- 将 BDrawing 元素数据字段统一为 `name`，移除元素类别、形状联合类型与工具模式类型定义，并让图层列表按元素注册 `label` 展示标题。
- 优化画图侧边栏交互反馈，工具项支持 hover 背景变化，图层列表可跟随画布选区高亮。
- 调整画图右侧设置栏：有选中元素时展示对应元素注册的 `Setter.vue`，无选中元素时展示全局设置。
- 参考 AreaAction 结构重构画图右侧设置栏，将其收敛为按选中元素调度 Setter 的独立 action 容器。
- 将画图右侧设置栏页签切换改用 `a-tabs` 组件，保留单选显示设计/属性、多选仅显示设计的规则。
