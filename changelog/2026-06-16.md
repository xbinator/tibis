# 2026-06-16

## Added
- 为 BMessage Markdown 图片增加悬浮复制按钮，支持复制图片本体到系统剪贴板。
- 为聊天任务列表完成态增加折叠摘要与关闭入口。
- 为 BDrawing 画板增加撤销与重做历史快捷键支持。
- 为 BDrawing 打开已有内容时增加首屏自动适配视图。
- 增加 AI 读取与更新当前 BDrawing 画板的内置工具。
- 增加 AI 按操作协议增删改 BDrawing 画板元素与连线的内置工具。
- 增加 AI 创建 BDrawing 画板草稿的内置工具，支持创建时同步应用初始画板操作。

## Changed
- 简化 BMessage Markdown 图片复制按钮样式，移除投影与额外焦点描边。
- 聊天任务运行中自动展示任务面板，避免未完成任务被折叠隐藏。
- 调整聊天任务全部完成或取消后的处理逻辑，立即清理持久化任务并隐藏任务面板。
- 优化聊天回到底部按钮的显示时机，滚动时立即显示，停止滚动后延迟隐藏。
- 重构 BDrawing 画板快捷键实现，统一复用 useShortcuts 注册与清理能力。
- 调整 BDrawing 模型同步，视图平移和缩放不再触发文件待保存状态。
- 收紧 AI 操作 BDrawing 画板工具的参数 Schema，方便模型稳定调用结构化操作。
- 降级 AI 完整替换 BDrawing 画板工具的默认暴露，避免模型绕过结构化操作协议。
- 优化 AI 创建 BDrawing 连线的数据与锚点逻辑，避免重复旧连线渲染并减少默认连线穿节点。
- 优化 BDrawing 默认连接线路由，使用正交折线、正确箭头方向与折线路径标签位置提升流程图可读性。
- 优化 BDrawing 连接线基础避让逻辑，默认折线遇到阻挡节点时优先选择安全绕线路径。
- 移除 BDrawing 旧版 edge 连线渲染与类型依赖，画板状态统一丢弃旧 edge 数据并仅保留新版 connector。
- 优化 BDrawing 近似同轴连接线路由，端点非常接近水平或垂直对齐时自动吸附为单段直线。
- 收窄 BDrawing 近似同轴连线吸附阈值，减少稍有偏移时被误拉直的情况。
- 优化 BDrawing 视口移动时 Selecto 与 Moveable 选区覆盖层的刷新时机，减少选中框滞后。
- 调整 BDrawing 选区覆盖层在视口变化后的刷新节奏，等待下一帧布局完成后再重新测量位置。
- 优化 BDrawing 同端点多条连接线的路由表现，自动错开平行连接线避免完全重叠。
