# 2026-06-12

## Added
- BDrawing 选中节点后支持在左侧样式面板修改背景色、描边色和描边粗细。
- BDrawing 连接线工具支持拖拽创建、创建预览和四向连接锚点。
- BDrawing 选中连接线后支持修改描边色、描边粗细、起点/终点箭头和直线/贝塞尔路径。

## Changed
- BDrawing 节点样式面板改为内置色板和描边宽度按钮，并避免点击面板时清空当前选中节点。
- BDrawing 节点颜色改为同步到 SVG 内联样式，修复选中状态下背景色和描边色视觉上不更新的问题。
- BDrawing 矩形节点改为直角渲染，不再显示圆角。
- BDrawing 样式面板颜色项改为使用十六进制输入框，并与描边宽度按钮统一为 28px 高度。
- BDrawing 节点选中控制框增加 8px 外扩留白，避免选框紧贴节点边缘。
- BDrawing Moveable 控制点改为圆角正方形样式，增强选中控制点的边界感。
- BDrawing 矩形创建预览和小地图缩略图改为直角渲染，与画布节点保持一致。
- BDrawing 样式面板内复制颜色输入框内容时不再触发画板快捷键导致选区隐藏。
- BDrawing 移除画板根级键盘快捷键，工具切换、撤销重做和缩放统一通过工具栏操作。
- BDrawing 创建工具激活时展示样式面板，并将面板样式应用到下一个新建形状。
- BColorPicker 输入有效颜色时立即同步变更，修复 BDrawing 创建前输入颜色未失焦导致新形状未应用颜色的问题。
- BDrawing 恢复 Delete/Backspace 删除选中元素，并避免输入框内删除内容时误删画板元素。
- BDrawing 修复节点上松开鼠标时直接拖拽未结束，导致节点继续跟随鼠标移动的问题。
- BDrawing 连接线结束时增加按释放坐标命中目标节点的兜底，修复拖拽结束事件未落在目标节点上时无法创建连线的问题。
- BDrawing 连接线支持点击选中和删除，并在节点拖拽预览时同步更新关联连接线路径。
- BDrawing 连接线选中样式改为轻量短虚线，不再呈现大块高亮效果。
- BDrawing 连接线选中时高亮起点和终点锚点，增强连接关系识别。
- BDrawing 调整连接线选中端点的渲染层级，使端点高亮显示在节点上方且线条仍保持在节点下方。
- BDrawing 将连接线箭头改为节点上方的独立标记层，修复贝塞尔曲线终点箭头被目标节点遮挡的问题。
- BDrawing 连线样式面板的曲线和箭头选项改为图标按钮，不再显示文字选项。
- BDrawing 修复节点拖拽预览时连接线箭头未跟随更新，以及贝塞尔曲线终点箭头方向反向的问题。
- BDrawing 创建连接线时 hover 到目标节点会展示激活样式，增强可连接目标反馈。
- BDrawing 创建连接线时的目标反馈改为激活对应连接锚点，不再高亮整个节点。
- BDrawing 将连接线可见层和创建预览层提升到节点上方，并增强连接锚点激活时的可见反馈。
- BDrawing 收窄连接线端点模板类型，消除 Vue TS 插件对可空端点坐标的误报。
- MCP 远程 server 配置支持解析并传递自定义 headers，兼容 `type: "streamablehttp"` 配置格式。
- BColorPicker 弹层样式改用当前主题变量，修复背景、边框和阴影显示异常。
- BColorPicker 触发区改为快捷预设色与自定义颜色入口，并将颜色输入框移入紧凑下拉面板。
- BDrawing 样式面板宽度改为跟随内容实际尺寸，不再固定为 220px。
- BColorPicker 下拉面板改为从触发区左上方展开。
- WebView 截图操作改为复制 PNG 图片到系统剪贴板，不再弹出文件保存流程。
