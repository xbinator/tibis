# 2026-07-02

## Changed
- 将 Widget JS 脚本编辑从设置面板弹窗迁移到当前页面内的 `CodeEditor` 编辑器，并通过双向绑定实时写回脚本。
- `CodeEditor` 迁移到 `src/views/widget/components/CodeEditor.vue`，并在顶部工具栏新增“运行代码”标题。
- `CodeEditor` 改为覆盖层展示，打开编辑器时保留 Widget 设计页组件实例，顶部仅保留关闭按钮。
- Widget 文件保存前会先同步最新业务数据到文件内容，避免保存时写入旧的 `.tibis` 内容。
- `CodeEditor` 中所有 “JS 脚本” 文字统一替换为 “运行代码”（含标题、文件头描述及 JSDoc 注释）。
- Widget 运行代码入口从 `defineConfig({ ... })` 改为 `Widget({ ... })`，旧入口不再兼容。
- 将 Widget 运行态脚本适配逻辑合并回 `widgetRuntime.ts`，并把 HTTP 桥接抽为通用沙箱宿主能力。
- `runPartSandbox` 与 Widget 完成态创建改为对象参数，减少可选位置参数带来的阅读成本。
- 托管 request 对无效 JSON 响应返回原始文本，由业务方根据 `ok/status/data` 自行处理。
- 托管 request 响应超过读取上限时停止继续读取并返回已读取文本，不再抛出响应过大错误。
- Widget 运行态公开 API 从 state 体系迁移到 data 体系，支持 `Widget({ data })`、`this.$data`、`this.$setData` 和 `{{ data.xxx }}`。
- 创建小组件时会在表单校验阶段拦截重复标识，避免覆盖已安装小组件。

## Removed
- 移除 Widget JS 脚本弹窗编辑器。
