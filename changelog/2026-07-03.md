# 2026-07-03

## Added
- BSectionBlock 与 BSectionItem 新增 `labelMinWidth` 配置，支持统一设置或单项覆盖文字和图标前缀的最小宽度。
- Widget 元素属性面板新增高级循环数据配置，支持单元素与组合元素按数组数据源展开渲染，并提供循环变量绑定候选。

## Changed
- 移除聊天消息列表的 `v-memo` 渲染记忆，避免 open_widget 内嵌小组件运行态数据更新后视图被缓存依赖阻断。
- 修复 Monaco 编辑器聚焦时 `Ctrl/Cmd+S` 被空命令吞掉的问题，保存快捷键现在会转发到编辑器和 Widget 文件保存流程。
- 将 `src/components/BChat/components/MessageBubble/` 下 8 个 `BubblePart*.vue` 平铺文件重构为文件夹结构（`BubblePart*/index.vue`），同步更新 19 处 import（含 1 处 less `@import`），git 识别为 rename 保留文件历史。tsc / eslint / stylelint 全部通过。
- 将小组件运行态脚本执行工具迁移到 `src/components/BWidget/utils/widgetRuntime.ts`，由 `src/components/BWidget/Runtime.vue` 内部执行 mounted、interaction 与 submit 收尾，并通过 `change` 事件向外返回执行结果。
- 修正 BChat BubblePart 相关测试中的旧组件导入路径，避免文件夹化后测试在 import-analysis 阶段失败。
- 修复小组件运行态连续交互在宿主消息回写前复用旧状态的问题，`BWidgetRuntime` 现在会串行执行脚本并使用本地运行态快照衔接 props 回写。
- 精简 Widget 页面脚本配置工具引用，移除视图层纯转发文件、未使用常量转发与单点包装函数。
- 修复 Widget 循环数据展开的运行态布局，动态文本按每次迭代的局部上下文测量，并保持展开元素在原始层级槽位渲染。
- 重构 Widget 高级循环配置面板，数据源候选统一来自 `useElementVariables`，循环元素改为 `v-model:elements` 双向同步，并将默认变量名存储为空值。
- 移除 `src/` 与 `test/` 中的测试专用 DOM 标识属性，相关测试改用 class、组件 stub 和用户可见结构定位。
- 修复 Widget 高级循环配置表单写回临时副本的问题，字段变更会同步到 `elements` model，并继续拦截重复的循环变量名。
- 移除 Widget 节点和 WebView 自动化中的 `data-widget-*` / `data-test` 属性依赖，Widget DOM 命中改为内部映射注册。
- 重构 Widget 组合数据模型，组合改为 `name: 'group'` 的真实嵌套元素树，子元素使用父级相对坐标，并同步更新编辑、运行态、循环、侧栏与测试。
- 补齐 Widget 嵌套组合编辑行为，侧栏图层支持递归子元素操作，层级排序、复制粘贴和批量面板改为同父级与父级局部坐标语义。
- 修复 Widget 默认运行脚本缺少 `message` 初始 data 字段的问题，使默认变量候选与脚本摘要保持一致。
- 修复 Widget 嵌套组合选区、粘贴与标题递增边界：父子同时命中时保留祖先，选中组合后粘贴进组合内部，新建元素标题会统计嵌套子元素。
- 修复 Widget 直接拖拽组合时子元素预览不跟随的问题，父元素移动过程中会同步偏移整棵子树。
- 修复 Widget 缩放组合时子元素预览不跟随的问题，父元素尺寸调整过程中会同步更新直接子元素的位置和尺寸。
- 修复 Widget 通过 Moveable 控制框移动组合时子元素预览不跟随的问题，移动过程会同步更新组合子树预览。
- 修复 Widget 移动组合时子元素被当作吸附参考线的问题，选中组合的整棵子树会从 Moveable guideline 中排除。
- 优化 Widget 侧栏图层树的层级展示，并支持通过拖拽将图层移入组合或从组合内移出。
- 优化 Widget 侧栏图层拖拽提示线的层级缩进，并在组合被清空时自动移除空组合。
